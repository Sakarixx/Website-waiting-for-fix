ยินดีต้อนรับเข้าสู่เว็บไซต์
อาหารไทย เป็นสิ่งที่แสดงถึงภูมิปัญญาและเอกลักษณ์ของชนชาติไทย เสน่ห์ของอาหารไทย นั้นมีเอกลักษณ์ที่แตกต่างกันไปในแต่ละสูตรอาหาร ซึ่งส่วนผสมที่หลากหลายของการปรุงรสชาติของอาหารไทย ผสมผสานจนเกิดความอร่อยและมีคุณค่าทางโภชนาการที่สูง

โดยแต่ละภาค จะมีความแตกต่างของอาหารแต่ละภาค โดยอาจจะแตกต่างทางวัฒนธรรม ประเพณี ความเชื่อ และวัตถุดิบที่หาทานได้งานในภูมิภาคนั้น

โดยวันนี้จะยกตัวอย่างอาหารในแต่ละภาคมาอย่างละ 2 ชนิด
